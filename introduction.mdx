## What is SmartSpin2k?

SmartSpin2k is an open-source DIY project that transforms any spin bike with a resistance control knob into a fully-featured smart trainer. Built around an ESP32 microcontroller and a stepper motor, SmartSpin2k automatically controls your bike's resistance, allowing you to connect to popular training apps like Zwift, TrainerRoad, and Rouvy.

## How It Works

SmartSpin2k uses a stepper motor mounted to your spin bike's resistance knob. When you're riding in Zwift or another training app:

1. **The app sends resistance commands** via Bluetooth Low Energy (BLE) using standard FTMS protocol
2. **SmartSpin2k receives the command** and calculates the required knob position
3. **The stepper motor turns the knob** to match the target resistance or power output
4. **Your bike's resistance changes automatically** as you climb hills, enter ERG mode workouts, or race with others

<Note>
  SmartSpin2k works with any spin bike that has a manual resistance knob - no need for an expensive smart trainer!
</Note>

## Key Capabilities

### Automatic Resistance Control
SmartSpin2k responds to incline commands from your training app, automatically adjusting your bike's resistance to simulate hills and terrain changes. This creates an immersive training experience without manual intervention.

### ERG Mode for Structured Workouts
In ERG mode, SmartSpin2k maintains a target power output by automatically adjusting resistance as your cadence changes. This is essential for structured interval training and following power-based workout plans.

From `ERG_Mode.h:26`:
```cpp
void runERG();      // Main control function for ERG Mode
void computeErg();  // Calculate resistance adjustments
```

### Power Calibration System
Many spin bikes have inaccurate power meters. SmartSpin2k includes a power table calibration system that learns your bike's actual power curve and provides accurate power readings.

From `Power_Table.h:25`:
```cpp
// Pick up new power value and put them into the power table
void processPowerValue(PowerBuffer& powerBuffer, int cadence, Measurement power);

// returns target position for given cadence and watts
int32_t lookup(int watts, int cad);
```

### Bluetooth Connectivity
SmartSpin2k broadcasts multiple BLE services simultaneously:
- **FTMS (Fitness Machine Service)** - Standard protocol for training apps
- **Cycling Power Service** - Power meter data
- **Cycling Speed & Cadence** - Real-time metrics
- **Heart Rate Service** - Pass-through from connected HR monitors

## Open Source Hardware & Software

SmartSpin2k is completely open source:
- **Hardware**: [CERN Open Hardware License v2](https://github.com/doudar/SmartSpin2k/blob/develop/Hardware/LICENSE)
- **Software**: [GNU GPL v2](https://github.com/doudar/SmartSpin2k/blob/develop/LICENSE)
- **Platform**: ESP32 with Arduino framework
- **Build System**: PlatformIO

<Tip>
  The entire build process takes under an hour with basic tools and a 3D printer. All STL files and PCB designs are freely available.
</Tip>

## What You Need

To build SmartSpin2k, you'll need:

- A spin bike with a resistance control knob
- 3D printer access (or purchase pre-printed parts)
- ESP32 development board
- NEMA 17 stepper motor
- TMC2208 or TMC2209 stepper driver
- Basic soldering skills
- Standard tools (screwdriver, wire strippers)

See the [Quick Start guide](/quickstart) for detailed requirements and assembly instructions.
